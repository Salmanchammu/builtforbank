All Global Colors (Cheat Sheet)

A complete guide to the :root variables in modern-dashboard-fixed-v2.css.

1. Primary Brand Colors

Change these if the lecturer asks to change the "Theme" of the bank.

--primary-maroon: #9b1c31;
The signature SmartBank red. Controls active navigation items on the sidebar, header logos, and primary accent highlights.
--primary-blue: #3b82f6;
The primary action color. Controls main interactive buttons (like "Transfer Money"), glowing hovers, and progress bars.
--gold-accent: #d4af37;
Premium gold accent used for VIP/Premium UI elements and card underlines.

2. Background Colors

Change these if the lecturer asks to change the page color (e.g., from light gray to white or dark).

--bg-gradient-start: #F4F7FE;
The soft cool gray that makes up the entire background of the application outside of the white cards.
--card-bg: #FFFFFF;
The crisp white background used for all dashboard widgets and data tables.

3. Text Colors (Typography)

Change these if the text is hard to read or you switch to a darker background.

--text-primary: #0f172a;
Almost black. Used for all main headings, titles, and important numbers (like account balances).
--text-secondary: #475569;
Dark gray. Used for regular paragraph text, table headers, and descriptions.

4. State & Alert Colors

Used specifically for badges and status indicators. Never use these for primary branding.

--success: #059669;
Green. Used for "Approved" loans, successful transfers, and active accounts.
--danger: #e11d48;
Bright Red. Used for "Rejected" applications, error messages, and delete/logout buttons.
--warning: #d97706;
Orange/Yellow. Used for "Pending" statuses.
Viva Tactic: If the lecturer asks "How do you handle Dark Mode?"
Answer: "I have a specific CSS block called body.dark-theme. When a user clicks the moon icon, JavaScript adds the 'dark-theme' class to the body. This CSS block completely overrides these :root variables, swapping the white backgrounds for dark navy blues and the dark text for white text instantly, without reloading the page."
