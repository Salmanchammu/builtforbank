How to Instantly Change UI Colors in the Viva

If the lecturer asks you to change the color of the buttons or the overall theme during your presentation, DO NOT hunt through thousands of lines of CSS to change individual buttons. Use this exact trick to demonstrate your knowledge of modern CSS architecture.

Method 1: The VS Code Approach

  1. Open your main CSS file: frontend/css/modern-dashboard-fixed-v2.css.
  2. Scroll to the very top (around Line 7). You will see the :root block.
  3. This block acts as a global "Control Panel" for all colors in your app. Find Line 35:
:root {
    ...
    --primary-blue: #3b82f6;
}
  1. Change the hex code to whatever the lecturer wants (e.g., #10b981 for green).
  2. Save the file and refresh the browser. All primary buttons, active navigation items, borders, and hover effects will instantly update to the new color!

What to say to the Lecturer:

"Instead of hardcoding colors on every individual button, I engineered a centralized Design System using CSS `:root` Variables (Custom Properties). By changing the variable here at the top of the file, the browser automatically cascades that change down to every single button, icon, and border in the application. This strictly follows the DRY (Don't Repeat Yourself) principle and makes the application incredibly easy to theme."

Method 2: The "Live Browser" Approach (Highly Impressive)

If you don't want to open VS Code at all, you can change it live right in front of them!

  1. Right-click anywhere on the dashboard background and select "Inspect" (or press Ctrl+Shift+I).
  2. In the Elements panel, click on the top <html> tag.
  3. Look at the Styles box on the right side. You will see your :root { ... } variables listed with little colored squares next to them.
  4. Click the colored square next to --primary-blue. A color picker tool will pop up!
  5. Drag the color picker to the new color. The entire website's buttons will change colors live in real-time as you drag your mouse!
Why this works: This proves to the examiner that you deeply understand how the CSS DOM operates in real-time within the browser engine, and that your codebase is perfectly modular.
